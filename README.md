# Адресная книга

Простой справочник на основе `Redis` + `FastAPI` хранящий связки телефон:адрес

## Локальный запуск проекта через docker

- запустить `docker-compose build`
- запустить `docker-compose up`
- перейти на `http://localhost:8000/docs` для проверки работы

## Переменные окружения

- `REDIS_HOST` - адрес хоста `redis` для подключения (`str`)
- `REDIS_PORT` - порт хоста `redis` для подключения (`int`)
- `REDIS_DB` - номер базы `redis` для подключения (`int`)
- `REDIS_TTL` - время жизни ключа в `redis` (`TTL индекс`) (`int`)
